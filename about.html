About Ecliptik

Idea
Ecliptik is a gender symbol which is used to express sex, gender and sexual orientation. Contrary to conventional symbols, it does not use a binary distinction between the two poles like ♂ Mars and ♀ Venus. Instead, any value between those poles can be used for more accurate expression of one's own gender identity, sex and orientation. But the biggest achievement of this symbol is that it does not need 3 symbols for the full representation of one's sex ⚧, gender identity ♂ and sexual orientation ⚤. Yet, it provides a simple representation that is easy to interpret at the first look.

About us
Ecliptik was invented in 2018 by two computer science students from Austria, Mathias Parger and Philipp Reitter. It is an open source project using the MIT licence and the created images use the Creative Commons licence CC0, i.e. they are free to use and edit for every person and company. Our goal is to replace conventional gender symbols by Ecliptik where possible. Therefore, we are providing easy to implement tools for websites and we are working on modules for additional platforms to make the integration as easy as possible.
